## 🔄 **자동 백업 시스템 (Velog → GitHub)**
이 저장소는 **GitHub Actions를 사용하여 매일 오전 9시 자동 업데이트**됩니다.  
즉, Velog에 글을 작성하면 다음날 자동으로 `.md` 파일로 저장됩니다.
